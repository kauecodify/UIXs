# Spotily Backend

Backend Node.js para download de músicas do YouTube e conversão para MP3.

## 🚀 Como funciona

O backend recebe uma URL do YouTube via API, usa o **yt-dlp** para baixar o áudio e converter automaticamente para MP3, salvando na pasta `/downloads`. O frontend então recebe a URL do arquivo para tocar no player.

## 📦 Instalação

### Pré-requisitos
- **Node.js** instalado
- **yt-dlp** instalado (https://github.com/yt-dlp/yt-dlp)
- **ffmpeg** instalado (para conversão de áudio)

### Passos js
```bash
# Instalar dependências
npm install

# Rodar o servidor
